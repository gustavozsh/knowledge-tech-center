# Research Papers

This directory contains research papers, whitepapers, and academic publications.

## Structure

Organize papers by topic:

```
papers/
├── machine-learning/
├── distributed-systems/
├── databases/
├── security/
├── cloud-computing/
└── software-engineering/
```

## Adding Papers

When adding a paper, consider:

1. **File naming**: `YYYY-author-title.pdf` (e.g., `2023-google-attention-is-all-you-need.pdf`)
2. **Include a reference**: Add entry to a references file or README
3. **Cite the source**: Include DOI or URL when available
