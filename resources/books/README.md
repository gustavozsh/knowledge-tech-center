# Technical Books

This directory contains references and notes from technical books and textbooks.

## Structure

Organize by topic:

```
books/
├── programming/
├── system-design/
├── data-engineering/
├── machine-learning/
├── devops/
└── cloud/
```

## Adding Book References

When adding book content:

1. **Create a directory** for each book with notes and summaries
2. **Include metadata**: Title, Author, ISBN, Year
3. **Respect copyright**: Only share notes, summaries, or open-source content
