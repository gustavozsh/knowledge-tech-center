# Technical Articles

This directory contains links and notes from technical articles, blog posts, and case studies.

## Structure

Organize by topic:

```
articles/
├── architecture/
├── best-practices/
├── case-studies/
├── tutorials/
└── news/
```

## Adding Articles

When adding an article:

1. **Create a markdown file** with notes and the source link
2. **Include metadata**: Title, Author, Source, Date
3. **Add key takeaways** and summaries

## Article Template

```markdown
# Article Title

**Source**: [Link](url)
**Author**: Name
**Date**: YYYY-MM-DD
**Tags**: tag1, tag2

## Summary

Brief summary of the article.

## Key Points

- Point 1
- Point 2
- Point 3

## Notes

Additional notes and thoughts.
```
